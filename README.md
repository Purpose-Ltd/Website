# Purpose – Next-Generation Recruiting Technology

Purpose is a deep tech recruiting platform that combines proprietary matching algorithms, AI-powered candidate screening, and automated application technology.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** GitHub Pages

## Pages

- `/` – Main landing page
- `/about` – About us
- `/cta` – Call to action
- `/privacy-policy` – Privacy Policy
- `/terms-and-conditions` – Terms & Conditions
- `/imprint` – Imprint

## Development

Install dependencies:
```
yarn install
```

Run local development server:
```
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

The website is automatically deployed to [purpos3.de](https://purpos3.de) via GitHub Actions on every push to `main`.
